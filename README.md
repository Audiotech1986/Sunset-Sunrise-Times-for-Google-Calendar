\# Sunrise/Sunset Calendar Generator



A GUI tool to generate Google Calendar CSV files for sunrise and sunset times.



\## Features

\- Enter a location (city, town) and generate sunrise/sunset timings.

\- Produces a CSV file ready to import into Google Calendar.

\- GUI built with Python `tkinter`.

\- Optional custom icon for Windows executable.



\## Requirements

\- Python 3.8+

\- Packages: `requests`, `pytz`

\- `tkinter` (included in standard Python)
